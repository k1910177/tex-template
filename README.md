# Latex Template

## Requirements

- VsCode
- Docker

## Instructions

Clone this repository

```sh
git clone https://github.com/k1910177/tex-template sample-project
```

Open in VsCode

```sh
cd sample-project
code .
```

Edit container name in `devcontainer.json`

Open workspace in devcontainer and wait for it to boot up

Compile all

```sh
make
```
